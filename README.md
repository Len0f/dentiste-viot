# Dentiste Viot — Site vitrine

Site vitrine pour un cabinet dentaire. Projet client en cours de finalisation, en attente des photos pour compléter l'intégration visuelle.

Demo : [dentiste-viot.vercel.app](https://dentiste-viot.vercel.app)

## Technologies

- **Framework** : Next.js 12
- **UI** : React 18, Font Awesome
- **Tests** : Jest, Testing Library
- **Langage** : JavaScript
- **Déploiement** : Vercel

## Statut

Projet en cours — en attente des photos du cabinet pour finalisation.

## Fonctionnalites

- Présentation du cabinet et de l'équipe
- Informations sur les soins proposés
- Coordonnées et accès
- Design responsive

## Structure du projet

- `components/` — Composants React réutilisables
- `pages/` — Pages Next.js (routing automatique)
- `public/` — Fichiers statiques
- `styles/` — Fichiers CSS
- `jest.config.js` — Configuration des tests
- `next.config.js` — Configuration Next.js
